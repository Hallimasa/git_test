Hypertext Markup Language - HTML

  • Controls the layout of the content
  • Provides structure for the webpage design
  • The Fundamental building block of any webpage

Semantic HTML

HTML has a vast list of predefined tags that you can use to create all kinds of different elements. It is important to use the correct tags for content. Using the correct element for content is called semantic HTML

Nesting and Indentation

You may have noticed that in all the examples in this lesson we indent any elements that are within other elements. This is know as nesting elements. When we nest elements within other elements, we create a parent and child relationship between them. The nested elements are the children and the element they are nested within is the parent

The parent, child and sibilins relationships between elements will become much more important later when we start styling our HTML with CSS and adding behavior with JavaScript

Text formative Elements

  • <b> - Defines bold text with no importance
  • <em> - Defines emphasized text
  • <i> - Defines a part of text in a alternative voice or mood
  • <small> - Defines smaller text
  • <strong> - Defines important text
  • <sub> - Defines subscripted text
  • <sup> - Defines superscripted text
  • <ins> - Defines inserted text
  • <del> - Defines deleted text
  • <mark> - Defines marked/highlighted text
